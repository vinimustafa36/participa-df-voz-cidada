# Participa DF – Voz Cidadã

Protótipo de Progressive Web App (PWA) acessível para o registro de manifestações de ouvidoria no âmbito do Governo do Distrito Federal.

## Visão Geral

O Participa DF – Voz Cidadã é uma solução digital concebida para ampliar o acesso da população aos canais de participação social, permitindo o envio de manifestações por múltiplos formatos, com foco em acessibilidade, simplicidade e empatia institucional.

Este repositório contém o código-fonte e a documentação do protótipo desenvolvido para o 1º Hackathon em Controle Social – Desafio Participa DF.
