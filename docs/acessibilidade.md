# Diretrizes de Acessibilidade

O projeto Participa DF – Voz Cidadã foi desenvolvido em conformidade com as diretrizes de acessibilidade digital WCAG 2.1 AA.

## Princípios Adotados

- Interface navegável por teclado
- Compatibilidade com leitores de tela
- Uso de alto contraste
- Textos simples e objetivos
- Elementos interativos com rótulos claros
- Feedback visual e textual ao usuário

## Abordagem

A acessibilidade foi considerada desde a concepção da solução, não como um complemento, mas como um requisito essencial para garantir inclusão digital e equidade no acesso aos serviços públicos.
